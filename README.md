## 🤍 시작 가이드
### Backend Requirements
- **spring Framework** : 6.1.13
- **JDK** : 21
- **Build Tool** : Gradle

**IntelliJ IDEA** : MainApplication Run 버튼 실행 or 아래 커맨드를 통해 빌드 & 실행
```
git clone https://github.com/M-bridge-3rd-9oormthonUNIV/Back.git
cd Back
./gradlew build
./gradlew bootRun
```
